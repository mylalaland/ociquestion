# AI Quiz Generator - Development Log (2026-04-14)

## 📌 개요
오늘 작업은 사용자가 겪고 있던 Gemini AI 연결 문제 해결을 시작으로, 문서 분석의 정확도를 높이고 한국어 환경에 최적화된 결과물(PDF)을 생성하는 전 과정을 고도화했습니다.

---

## ✅ 주요 해결 과제 및 성과

### 1. AI 연결성 및 안정성 확보 (AI Stability)
- **404 Model Not Found 해결**: 사용자의 API 키가 접근 가능한 실제 모델 목록을 실시간으로 가져오는 **'모델 조회(Discovery)'** 기능을 구현했습니다.
- **503 Server Busy 해결**: Google 서버 부하 시 포기하지 않고 지연 시간을 늘려가며 재시도하는 **'지수 백오프(Exponential Backoff)'** 로직을 적용했습니다.
- **API 키 보안 및 위생**: 비가시적 특수문자로 인한 Header 에러를 방지하기 위해 입력을 엄격히 정제하는 로직을 강화했습니다.

### 2. 멀티모달 입력 시스템 고도화 (Multi-Modal)
- **멀티 샷 카메라 갤러리**: 한 장씩만 찍을 수 있던 한계를 넘어, 여러 장의 교재 페이지를 연속 촬영하고 검토할 수 있는 카메라 시스템을 구축했습니다.
- **배치 프로세싱**: 여러 개의 PDF와 이미지 파일을 한 번에 분석하여 통합된 퀴즈를 생성하도록 로직을 확장했습니다.

### 3. 사용자 맞춤형 퀴즈 설정 (Customization)
- **퀴즈 구성 패널**: 문제 생성 전 **문제 수(1~20개)**와 **문제 유형(수능형, 객관식, 단답형)**을 직접 선택할 수 있는 UI를 추가했습니다.
- **유형별 프롬프트 최적화**: 선택한 유형에 따라 문제 스타일과 난이도가 조정되도록 AI 지시문을 정교화했습니다.

### 4. 핵심 버그 수정 및 품질 개선 (Bug Fixes)
- **PDF 한글 깨짐 완전 해결**: 폰트 호환성 문제를 근본적으로 해결하기 위해 **화면 캡처 방식(html2canvas)**을 도입, 완벽한 한글 PDF 결과물을 보장합니다.
- **하이브리드 하이라이터**: AI의 응답과 원문 텍스트 사이의 미세한 오차를 무시하고 정확한 위치를 찾아주는 **유연한 텍스트 매칭**과 **자동 스크롤** 기능을 구현했습니다.
- **PDF 워커 경로 수정**: 외부 라이브러리(PDF.js)의 워커 파일을 안정적으로 불러오도록 CDN 경로를 최적화했습니다.
